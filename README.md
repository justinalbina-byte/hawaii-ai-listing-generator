# Hawaii AI Listing Generator

An AI-powered real estate listing generator for Hawaii properties, built with Python and the Anthropic Claude API.

## Features
- Generates professional MLS-style listing descriptions
- Calculates price per square foot automatically
- AI listing score and sellability rating out of 10
- Price competitiveness analysis for the Hawaii market
- Neighborhood report with walkability score
- Nearby attractions and neighborhood vibe summary
- Email-ready format for client outreach
- Saves every listing as a text file
- Multi-listing menu — generate multiple listings in one session

## Built with
- Python
- Anthropic Claude API
- python-dotenv

## How to run
1. Clone the repository
2. Install dependencies:
   pip3 install anthropic python-dotenv
3. Create a .env file in the project folder and add your Anthropic API key:
   ANTHROPIC_API_KEY=your-key-here
4. Run:
   python3 listing.py

## Example output
- AI-generated listing description
- Listing score and price analysis
- Neighborhood report with walkability score and nearby attractions
- Email-ready format saved to a .txt file
