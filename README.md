# Hawaii AI Listing Generator

An AI-powered real estate listing generator for Hawaii properties, built with Python and the Anthropic Claude API.

## Features
- Generates professional MLS-style listing descriptions
- Calculates price per square foot automatically
- AI listing score and sellability rating
- Price competitiveness analysis for the Hawaii market
- Email-ready format for client outreach
- Saves every listing as a text file
- Multi-listing menu — generate multiple listings in one session

## Built with
- Python
- Anthropic Claude API
- python-dotenv

## How to run
1. Clone the repository
2. Install dependencies: `pip3 install anthropic python-dotenv`
3. Add your Anthropic API key to a `.env` file
4. Run: `python3 listing.py`
